About the program

SHELF+ is an application for reading electronic books in PDF format. 
From the main menu, clicking on + will add books. When clicking on SHELF, you will go to the shelf. 
From the main menu, it is possible to display a report on account usage in a file ('Report.txt'), to print or send by e-mail. 
In the reading mode, it is also possible to print the book itself, change it and save it to disk.
